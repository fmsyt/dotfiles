# dotfiles

## Required packages

- git
- unzip

This repository has files encrypted by `git-secret`.
Install below if you needed and trusted.

- gnupg
- git-secret

## Installation

```bash
$ cd /path/to/anywhere
$ git clone https://github.com/fmsyt/dotfiles
$ ./dotfiles/install.sh
```

Reveal encrypted files if you have `git-secret` and `gpg`.

```bash
$ ./reveal.sh
```
